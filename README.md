# 🏡 Airbnb Data Analysis

This project focuses on data analysis of Airbnb listings data using datasets from Inside Airbnb.

## 📊 Project Overview

The goal of this project was to:
- Load and explore a large dataset
- Perform data cleaning and preprocessing
- Analyze numerical and categorical variables
- Handle missing values and outliers
- Discover relationships between variables
- Visualize data distributions and correlations

## 📁 Dataset

The dataset was obtained from:
http://insideairbnb.com/get-the-data.html

It contains detailed information about Airbnb listings for a Nashville, Tennese, United States.

## ⚙️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## 🔍 Key Steps

1. Data loading and inspection  
2. Data type verification and correction  
3. Missing values analysis  
4. Statistical summary of variables  
5. Data cleaning and preprocessing  
6. Outlier detection and handling  
7. Data visualization (histograms, scatter plots)  
8. Correlation analysis  

## 📈 Example Insights

- Price distribution is highly skewed
- Some variables contain significant missing data
- Strong correlations exist between selected numerical features
